# qaiik

- Next.js v16
- MDX Rust & Shiki inside React Server Components
- Tailwind CSS v4
- `<ViewTransition/>` from React Experimental

## SIFTLI Route

- App route: `/siftli`
- API route: `/api/siftli/send`
- Required env vars:
  - `TELEGRAM_BOT_TOKEN`
  - `TELEGRAM_CHAT_ID`
  - `DISCORD_WEBHOOK_URL`

Use `.env.example` as the template for local setup.
